๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ เ ก๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ ๏ฟฝ๏ฟฝ ใฆฅ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝเฎฅ๏ฟฝ๏ฟฝ ๏ฟฝ ๏ฟฝเฎฅ๏ฟฝ๏ฟฝ๏ฟฝ. ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝโ ญ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝเ ข๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝแจฉ ๏ฟฝ๋ฏฎ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ:


## ๏ฟฝ๏ฟฝ๏ฟฝๆจ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ

๏ฟฝ๏ฟฝแซฅ ๏ฟฝ๏ฟฝโ ญ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝโฎข แฎง๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ:

```powershell

dotnet ef migrations add InitialCreate --startup-project ..\TashGresOpto.UI

```

## ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ ๏ฟฝเจซ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ๏ฟฝ
