# Инструкция по установке TashGresOpto на новое устройство

## 📋 Системные требования

### Минимальные требования:
- **Операционная система:** Windows 10 (версия 1809 или новее) или Windows 11
- **Процессор:** Intel Core i3 или эквивалент (2 ГГц)
- **Оперативная память:** 4 ГБ RAM
- **Место на диске:** 500 МБ свободного места
- **Разрешение экрана:** 1400×900 (минимум)
- **Интернет:** для скачивания зависимостей (при первой установке)

### Рекомендуемые требования:
- **ОС:** Windows 11
- **Процессор:** Intel Core i5 или выше
- **RAM:** 8 ГБ или больше
- **Место на диске:** 1 ГБ
- **Разрешение экрана:** 1920×1080

## 🛠️ Вариант 1: Установка готового приложения (для пользователей)

### Шаг 1: Скачать исполняемый файл

1. Перейдите на https://github.com/mufasadullaev/optonew/releases
2. Скачайте последнюю версию `TashGresOpto-v1.0.zip`
3. Распакуйте архив в удобное место (например, `C:\Program Files\TashGresOpto`)

### Шаг 2: Установить .NET Runtime

**ВАЖНО:** Приложение требует .NET 8.0 Runtime

#### Автоматическая установка:
1. При первом запуске `TashGresOpto.UI.exe` Windows предложит установить .NET
2. Нажмите "Да" и следуйте инструкциям

#### Ручная установка:
1. Перейдите на https://dotnet.microsoft.com/download/dotnet/8.0
2. Скачайте **".NET Desktop Runtime 8.0"** для Windows
3. Выберите версию для вашей системы:
   - **x64** для 64-битной Windows (большинство современных компьютеров)
   - **x86** для 32-битной Windows (старые компьютеры)
4. Запустите установщик и следуйте инструкциям

### Шаг 3: Запустить приложение

1. Откройте папку с приложением
2. Запустите `TashGresOpto.UI.exe`
3. При первом запуске будет создана база данных SQLite

**Готово!** Приложение готово к работе.

## 💻 Вариант 2: Установка из исходного кода (для разработчиков)

### Шаг 1: Установить необходимые инструменты

#### 1.1 Установить .NET 8.0 SDK

1. Перейдите на https://dotnet.microsoft.com/download/dotnet/8.0
2. Скачайте **".NET 8.0 SDK"** (НЕ Runtime!)
3. Запустите установщик
4. После установки откройте командную строку и проверьте:
   ```cmd
   dotnet --version
   ```
   Должно показать: `8.0.x`

#### 1.2 (Опционально) Установить Visual Studio 2022

Если планируете разрабатывать или изменять код:

1. Скачайте Visual Studio 2022 Community (бесплатно):
   https://visualstudio.microsoft.com/downloads/

2. При установке выберите:
   - ✅ ".NET desktop development"
   - ✅ "Desktop development with C#"

**ИЛИ** можно использовать Visual Studio Code (легче):
1. Скачайте VS Code: https://code.visualstudio.com/
2. Установите расширение "C# Dev Kit"

#### 1.3 Установить Git

1. Скачайте Git для Windows: https://git-scm.com/download/win
2. Запустите установщик (можно оставить настройки по умолчанию)
3. Проверьте установку:
   ```cmd
   git --version
   ```

### Шаг 2: Клонировать репозиторий

Откройте командную строку (cmd) или PowerShell:

```cmd
cd C:\
git clone https://github.com/mufasadullaev/optonew.git
cd optonew\CSharp_Migration
```

### Шаг 3: Восстановить зависимости

```cmd
dotnet restore
```

Эта команда скачает все необходимые NuGet пакеты:
- Entity Framework Core 8.0
- SQLite
- System.Text.Encoding.CodePages
- И другие

### Шаг 4: Собрать проект

```cmd
dotnet build
```

Если сборка успешна, вы увидите:
```
Build succeeded.
    0 Warning(s)
    0 Error(s)
```

### Шаг 5: Запустить приложение

```cmd
cd TashGresOpto.UI
dotnet run
```

**Готово!** Приложение запустится.

## 🗄️ База данных

### Автоматическое создание

При первом запуске приложение:
1. Создаст файл `TashGresOpto.db` в папке приложения
2. Применит все миграции Entity Framework Core
3. Заполнит справочные таблицы начальными данными

### Ручное создание (если нужно)

Если база данных не создалась автоматически:

```cmd
cd C:\optonew\CSharp_Migration\TashGresOpto.Data
dotnet ef database update --project ..\TashGresOpto.Data --startup-project ..\TashGresOpto.UI
```

### Расположение базы данных

По умолчанию база данных создается в папке с приложением:
```
C:\optonew\CSharp_Migration\TashGresOpto.UI\TashGresOpto.db
```

## 📦 Создание установочного пакета

### Вариант A: Простая публикация (portable)

Создать папку со всеми файлами для копирования на другой компьютер:

```cmd
cd C:\optonew\CSharp_Migration\TashGresOpto.UI
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true
```

Результат будет в:
```
C:\optonew\CSharp_Migration\TashGresOpto.UI\bin\Release\net8.0-windows\win-x64\publish\
```

Скопируйте все файлы из этой папки на новый компьютер - приложение будет работать без установки .NET!

### Вариант B: Публикация с .NET Runtime (меньший размер)

Если на целевом компьютере уже установлен .NET 8.0:

```cmd
dotnet publish -c Release -r win-x64 --self-contained false
```

### Вариант C: Создание MSI установщика

Для создания профессионального установщика потребуется:

1. Установить WiX Toolset: https://wixtoolset.org/
2. Создать `.wixproj` файл проекта установщика
3. Собрать MSI пакет

## 🔧 Возможные проблемы и решения

### Проблема 1: "Не удается найти .NET runtime"

**Решение:**
```cmd
winget install Microsoft.DotNet.DesktopRuntime.8
```

Или скачайте вручную с https://dotnet.microsoft.com/download/dotnet/8.0

### Проблема 2: "Access Denied" при создании базы данных

**Решение:**
1. Запустите приложение от имени администратора
2. ИЛИ установите приложение в папку пользователя:
   ```
   C:\Users\ВашеИмя\TashGresOpto\
   ```

### Проблема 3: Кириллица отображается некорректно

**Решение:**
1. Убедитесь, что файлы проекта сохранены в UTF-8 BOM
2. Проверьте кодировку:
   ```cmd
   chcp 65001
   ```

### Проблема 4: "dotnet: command not found"

**Решение:**
1. Перезагрузите компьютер после установки .NET SDK
2. Проверьте PATH:
   - Нажмите Win+R, введите `sysdm.cpl`
   - Advanced → Environment Variables
   - Проверьте, что есть путь к .NET:
     `C:\Program Files\dotnet\`

### Проблема 5: Приложение не запускается (черное окно появляется и исчезает)

**Решение:**
Запустите через командную строку, чтобы увидеть ошибку:
```cmd
cd C:\путь\к\приложению
TashGresOpto.UI.exe
```

## 📝 Пошаговая инструкция для обычного пользователя

### Для пользователя без технических знаний:

1. **Скачать и распаковать**
   - Скачайте файл `TashGresOpto-Setup.zip`
   - Кликните правой кнопкой → "Извлечь все"
   - Выберите папку `C:\TashGresOpto`

2. **Установить .NET (если нужно)**
   - Запустите `TashGresOpto.UI.exe`
   - Если появится окно "Требуется .NET", нажмите "Да"
   - Дождитесь установки (5-10 минут)

3. **Запустить программу**
   - Снова запустите `TashGresOpto.UI.exe`
   - Программа готова к работе!

4. **Создать ярлык на рабочем столе**
   - Кликните правой кнопкой на `TashGresOpto.UI.exe`
   - "Отправить" → "Рабочий стол (создать ярлык)"

## 🌐 Сетевая установка (многопользовательская)

### Вариант 1: Общая папка с базой данных

1. Установите приложение на каждый компьютер
2. Создайте общую сетевую папку:
   ```
   \\server\TashGresOpto\Database\
   ```
3. Измените строку подключения в `appsettings.json`:
   ```json
   {
     "ConnectionStrings": {
       "DefaultConnection": "Data Source=\\\\server\\TashGresOpto\\Database\\TashGresOpto.db"
     }
   }
   ```

### Вариант 2: SQL Server (для будущего)

В будущих версиях можно перейти на SQL Server для лучшей многопользовательской работы.
